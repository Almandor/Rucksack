def rpgtoolbox.globaltools.findLoops (   struc = {},
  elem = '',
  way = [],
  result = [] 
)

This function looks for loops in a tree structure.

The given dictionary must have the following structure dic[struc_item]['subelem']. It runs recusively through the structure tree. Further it will change the global variable loops.

Parameters
structree in a dictionary which shall be searched for loop structures.
elemelement of the structure tree where to start the search for loops.
wayThe path which were actually run through in the tree.
resultcontains loops as a list of tuples.
Bug:
this function finds sometimes loops where no loops are... it seems to concern multiple links to leaves...
